どのアプリでも Cmd+C を2回押すだけで、コピーしたテキストが翻訳トーストとしてすぐに表示されます。アプリの切り替えも、貼り付け欄も、アカウントも不要です。

CCTrans はメニューバーに常駐し、必要なときだけ現れます。

使い方
• テキストを2回コピー(Cmd+C、Cmd+C) — 翻訳トーストがすぐ表示されます
• Shift+Cmd+2 — 画面をキャプチャして、見えているテキストを翻訳します
• メニューバーのアイコンから言語の選択、トーストの位置、リクエスト履歴を管理できます

標準でプライベート
標準の翻訳は Apple のオンデバイス翻訳で、すべて Mac の中だけで処理されます。テキストが端末の外に出ることはなく、開発者は何のデータも収集しません。解析ツールも、アカウントも、サーバーもありません。

最先端の AI モデルも使える(任意)
最新の LLM 翻訳を使いたい場合は、ご自身の OpenRouter API キーを登録してください。必要に応じて画面コンテキストのキャプチャを添えて、短く曖昧な文も前後の文脈どおりに翻訳します。

実用のために
• ブラウザ、エディタ、チャット、PDF — あらゆるアプリで動作
• トーストは指定した位置に表示され、自動で消えます
• 言語ペアは一度ダウンロードすればオフラインでも動作(Apple 翻訳)
• Dock アイコンのない軽量なメニューバーアプリ

権限について
• 入力監視(任意): 他のアプリでの Cmd+C 2回押しを検出します。許可しなくても、同じテキストを2回コピーすればクリップボード経由で翻訳されます。
• 画面収録(任意): スクリーンショット翻訳と画面コンテキストにのみ使用します。

CCTrans は無料のオープンソースアプリです。
