어떤 앱에서든 Cmd+C를 두 번 누르면 작업하던 화면 위로 번역 토스트가 바로 뜹니다. 번역 앱으로 갈아탈 필요도, 어딘가에 붙여넣을 필요도 없습니다.

CCTrans는 메뉴 막대에 조용히 머물다가 필요할 때만 나타납니다.

사용 방법
• 텍스트를 두 번 복사하면(Cmd+C, Cmd+C) 번역 토스트가 바로 나타납니다
• Shift+Cmd+2를 누르면 화면을 캡처해서 보이는 텍스트를 번역합니다
• 언어 선택, 토스트 위치, 요청 기록은 메뉴 막대 아이콘에 모여 있습니다

기본은 온디바이스
기본 번역은 Apple 온디바이스 번역이라 전부 Mac 안에서 처리됩니다. 복사한 텍스트는 기기 밖으로 나가지 않습니다. 개발자 쪽에는 분석 도구도 계정 시스템도 없고, 애초에 텍스트를 받을 서버가 없습니다.

원하면 최신 AI 모델로 (선택)
LLM 번역이 필요하면 본인 OpenRouter API 키를 넣으면 됩니다. 화면 캡처를 컨텍스트로 같이 보내는 옵션도 있는데, 한 줄짜리 모호한 문장을 번역할 때 차이가 큽니다.

실전용으로 만들었습니다
• 브라우저, 에디터, 메신저, PDF 어디서든 동작합니다
• 토스트는 정해 둔 위치에 떴다가 알아서 사라집니다
• 언어 쌍을 한 번 받아 두면 오프라인에서도 번역됩니다 (Apple 번역)
• Dock 아이콘 없이 메뉴 막대에만 있는 가벼운 앱입니다

권한 안내
• 입력 모니터링(선택): 다른 앱에서 누른 Cmd+C 두 번을 감지하는 데 씁니다. 허용하지 않아도 같은 텍스트를 두 번 복사하면 클립보드 방식으로 번역됩니다.
• 화면 기록(선택): 스크린샷 번역과 화면 컨텍스트 캡처에만 씁니다.

CCTrans는 무료 오픈소스입니다.
