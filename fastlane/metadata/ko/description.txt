어떤 앱에서든 Cmd+C를 두 번 누르면, 복사한 텍스트가 번역 토스트로 바로 떠오릅니다. 앱 전환도, 붙여넣기 창도, 계정도 필요 없습니다.

CCTrans는 메뉴 막대에 조용히 머물다가 필요할 때만 나타납니다.

사용 방법
• 텍스트를 두 번 복사(Cmd+C, Cmd+C) — 번역 토스트가 즉시 나타납니다
• Shift+Cmd+2 — 화면을 캡처해 보이는 텍스트를 번역합니다
• 메뉴 막대 아이콘에서 언어 선택, 토스트 위치, 요청 기록을 관리합니다

기본은 온디바이스, 끝까지 프라이빗
기본 번역은 Apple 온디바이스 번역으로 Mac 안에서만 처리됩니다. 텍스트가 기기 밖으로 나가지 않으며, 개발자는 아무 데이터도 수집하지 않습니다. 분석 도구도, 계정도, 서버도 없습니다.

원하면 최신 AI 모델로 (선택)
최신 LLM 번역을 쓰고 싶다면 본인의 OpenRouter API 키를 등록하세요. 필요할 때 화면 컨텍스트 캡처를 함께 보내, 짧고 모호한 문장도 주변 맥락에 맞게 번역합니다.

실전용으로 만들었습니다
• 브라우저, 에디터, 메신저, PDF — 모든 앱에서 동작
• 토스트는 지정한 위치에 떴다가 알아서 사라집니다
• 언어 쌍은 한 번만 내려받으면 오프라인에서도 동작 (Apple 번역)
• Dock 아이콘 없는 가벼운 메뉴 막대 앱

권한 안내
• 입력 모니터링(선택): 다른 앱에서 Cmd+C 두 번 누름을 감지합니다. 거부해도 같은 텍스트를 두 번 복사하면 클립보드 방식으로 번역됩니다.
• 화면 기록(선택): 스크린샷 번역과 화면 컨텍스트에만 사용됩니다.

CCTrans는 무료 오픈소스입니다.
