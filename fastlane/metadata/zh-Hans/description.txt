在任何应用里连按两次 Cmd+C，复制的文本立刻以翻译浮窗弹出 — 不用切换应用，不用粘贴框，也不用注册账号。

CCTrans 安静地待在菜单栏里，需要时才出现。

使用方法
• 把文本复制两次(Cmd+C、Cmd+C) — 翻译浮窗立即弹出
• 按 Shift+Cmd+2 — 截取屏幕并翻译画面上可见的文字
• 点击菜单栏图标即可选择语言、调整浮窗位置、查看请求记录

默认即私密
默认翻译完全通过 Apple 设备端翻译在你的 Mac 上完成。文本不会离开你的设备,开发者也不收集任何数据 — 没有统计分析、没有账号、没有服务器。

可选:用最强的 AI 模型
想要最前沿的大模型翻译?填入你自己的 OpenRouter API 密钥,CCTrans 就能调用顶尖 LLM,并可在需要时附带屏幕上下文截图,让简短、含糊的句子也能按上下文的真实含义翻译。

为真实工作而生
• 在所有应用中可用 — 浏览器、编辑器、聊天工具、PDF
• 浮窗出现在你指定的位置,并自动消失
• 语言包只需下载一次,之后离线可用(Apple 翻译)
• 轻量菜单栏应用:没有 Dock 图标,不添乱

权限说明
• 输入监控(可选):用于检测在其他应用中连按两次 Cmd+C。即使拒绝,把同一段文本复制两次也能通过剪贴板方式触发翻译。
• 屏幕录制(可选):仅用于截图翻译和屏幕上下文。
